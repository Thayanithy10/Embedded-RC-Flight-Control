EMBEDDED RC VEHICLE STABILIZATION SYSTEM 
Designed an embedded, real-time closed-loop flight control and stabilization system for an RC vehicle utilizing an ESP32 microcontroller and an MPU-series gyroscope to dynamically counteract mechanical instability and external disturbances.
•	Engineered a non-blocking hardware interrupt framework to capture high-resolution PPM/PWM input signals from an RC receiver across multiple channels simultaneously.
•	Developed a closed-loop control system utilizing a Proportional-Derivative (PD) algorithm to inject real-time yaw rate corrections into a directional drive mix matrix.
•	Implemented aggressive software-level fault isolation engines, including an I2C bus recovery routine to clear digital communication lockups and a custom slew-rate limiter to soften high-current motor surges.
•	Integrated a software-defined glitch filter and signal timeout watchdog to prevent channel pulse freezing, ensuring immediate fallback to safe neutral vectors during signal anomalies.
•	Conducted continuous live telemetry tracking via serial diagnostics to analyze multi-channel pulse-width modulation, gyro angular velocity deviations, and mixed motor outputs.
•	Leveraged hardware-level optimization insights from structural testing, implementing star-grounding layouts, low-ESR decoupling filtration, and electromagnetic shielding to isolate sensitive logic components from high-torque drivetrain noise.
•	Technologies Used: ESP32 Microcontroller, MPU Gyroscope, PWM/PPM Signals, Closed-Loop Control (PD), I2C Bus Architecture, Hardware Interrupts, Electrical Fault Isolation.
